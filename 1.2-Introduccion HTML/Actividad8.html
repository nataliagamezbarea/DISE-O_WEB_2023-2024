XML
Es un lenguaje de marcas que se usa para estructurar datos, almacenar y transportar datos.
HTML
Es un lenguaje de marcas que se utiliza para estructurar y desplegar una página web y sus contenidos.
W3C
Es el organismo mundial encargado de desarrollar tecnologías y protocolos que han hecho posible la Web.
RFC
Es un documento numérico en el que se describen y definen protocolos, métodos y programas de Internet.
